## WebScraper
用re库爬取豆瓣电影排行榜上的排名前250的电影排行,影片名信息，以及影片内容描述。然后通过我自己编写的格式把这些信息进行总结。
词云生成我使用了numpy（用于生成词云形状），jieba（可以将中文文档分成词汇）和wordcloud（词云可视化）库
